## center

Centers a element absolutely on any or multiple planes

* **Group:** General
* **Access:** public
* **Since:** 0.1.0

### Parameters

| Name        | Type                                                             | Description                     | Default      |
| :---------- | :--------------------------------------------------------------- | :------------------------------ | :----------- |
| `$position` | **[String](https://sass-lang.com/documentation/values/strings)** | The position or plane to center | `null` |

### Examples

```scss
.example {
  @include center();
}
```
